# 统院男篮记录系统

这是一个用于记录篮球比赛数据的Web应用程序，专为统院男篮设计。它提供了详细的个人和团队数据统计，并支持将数据导出为Excel文件。
# 注意，使用前先重置！！！！！！！！！！，否则换人得分等按钮无效
## 功能

- **球员管理:**
  - 添加和删除球员。
  - 设置首发阵容。
  - 比赛期间换人。
  - 一键添加预设的默认球员名单。

- **比赛数据记录:**
  - 记录两分球、三分球（命中和投失）。
  - 记录罚球（命中和投失）。
  - 记录进攻篮板和防守篮板。
  - 记录助攻、抢断、盖帽、失误和犯规。
  - 记录对手的得分、投篮、罚球、篮板和失误。

- **比赛控制:**
  - 实时更新比赛得分。
  - 切换比赛节次。
  - 撤销上一步操作。
  - 一键重置所有比赛数据。

- **数据导出:**
  - 将所有比赛数据（包括个人、球队和对手的详细统计）导出为Excel文件。

- **用户界面:**
  - 简洁的iOS风格界面。
  - 响应式设计，适配移动设备。
  - 通过标签页在比赛记录和球员管理之间切换。
  - 使用模态框（弹窗）来记录投篮、罚球、篮板和换人等具体操作。

- **数据持久化:**
  - 比赛数据会自动保存在浏览器的本地存储中，即使刷新页面也不会丢失。

## 使用说明

1.  **打开应用:** 直接在浏览器中打开 `统院男篮记录系统2.0.html` 文件。
2.  **球员管理:**
    - 在 “球员管理” 标签页下，您可以添加或删除球员。
    - 点击 “添加默认球员” 可以快速载入预设的球员名单。
3.  **开始比赛:**
    - 在 “比赛记录” 标签页下，从 “选择球员” 列表中点击一名球员，以选中他进行数据记录。
    - 点击相应的操作按钮（如 “2分球”、“篮板”、“助攻” 等）来记录数据。
    - 对于投篮和罚球，会弹出窗口让您选择 “命中” 或 “投丢”。
    - 对于篮板，会弹出窗口让您选择 “进攻篮板” 或 “防守篮板”。
4.  **换人:**
    - 点击 “换人” 按钮，在弹出的窗口中选择要换下和换上的球员，然后点击 “确认换人”。
5.  **记录对手数据:**
    - 在 “菜鸟对手数据” 部分，可以记录对手的得分、篮板和失误。
6.  **查看事件记录:**
    - “事件记录” 区域会显示最近的操作，您可以点击 “撤销” 来回退到上一步。
7.  **结束比赛与导出数据:**
    - 比赛结束后，点击 “导出Excel” 按钮，系统将生成一个包含所有统计数据的Excel文件。

## 技术栈

- HTML
- CSS
- JavaScript
- [SheetJS/xlsx](https://github.com/SheetJS/sheetjs) 用于生成Excel文件。
